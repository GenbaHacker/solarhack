# solarhack ☀️

太陽光発電システムの**提案・シミュレーション・経済性評価**を自動化するWebツール

## Features
- 🏠 **屋根面パネル配置図** - リアルタイム自動計算
- 🌍 **日射量シミュレーション** - METPV11 837観測所対応
- ⚡ **発電量試算** - 月別・年間発電量予測
- 📊 **収支評価** - キャッシュフロー・IRR計算
- 📄 **提案書生成** - PDF出力対応

## Performance
- First fetch: 44 KB (stations_lite.json)
- Second fetch: 3.3 KB (per-station JSON)
- 98% bandwidth reduction vs. full DB
